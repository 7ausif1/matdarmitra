# Matdar Mitra | Your Ultimate Voter Friend 🗳️🇮🇳

Matdar Mitra is a premium, AI-powered civic engagement platform designed to simplify the Indian electoral process. It empowers citizens with the knowledge and tools they need to participate in democracy with confidence.

## 🌟 Key Features

- **Interactive EVM & VVPAT Simulator**: A realistic, risk-free environment to practice casting a vote and verifying the paper trail.
- **AI Assistant (Powered by Google Gemini)**: A smart, natural language assistant that answers complex electoral queries in real-time.
- **Booth Finder (Google Maps)**: Helps users locate their nearest polling station and visualize booth locations.
- **The Voter's Journey**: A step-by-step visual guide to the polling day experience.
- **Knowledge Quiz**: An interactive test to reinforce understanding of democratic safeguards.

## 🛠️ Technology Stack

- **Frontend**: Vanilla HTML5, CSS3, and Modular JavaScript.
- **AI Engine**: Google Gemini 1.5 Flash API.
- **Mapping**: Google Maps JavaScript API.
- **Design**: Premium "Glassmorphism" aesthetic with responsive layouts.
- **Security**: Built-in XSS protection and defensive coding practices.

## 🎯 Chosen Vertical: Civic Engagement & Electoral Literacy

Our solution addresses the "Civic Assistant" persona, focusing on logical decision-making based on user context (location, registration status, etc.) and effective use of Google Services to provide real-world utility.

## 🚀 How to Run

1.  Clone the repository.
2.  Open `index.html` in any modern web browser.
3.  **Enable AI**: Click the settings icon in the "Assistant" section and enter your Gemini API Key.
4.  **Enable Maps**: Click the "Enable Google Maps" button in the Booth Finder section and enter your Google Maps API Key.

## 🧪 Testing

We have implemented a functional test suite to ensure logic reliability.
- To run tests, open the browser console on `index.html` and check the output, or check `js/test.js`.

## 🛡️ Security & Accessibility

- **Security**: All user inputs are sanitized using custom utility functions to prevent injection attacks.
- **Accessibility**: Includes high-contrast color palettes, semantic HTML, ARIA labels, and skip links for screen readers.
